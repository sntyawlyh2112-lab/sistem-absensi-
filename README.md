# Sistem Absensi Mahasiswa

## Anggota Kelompok
- Nazwa Aulia Fitri
- Santi Alawiyah

## Fitur Aplikasi
- Login
- Logout
- Dashboard
- Data Mahasiswa (Tambah, Edit, Hapus)
- Data Absensi (Tambah, Edit, Hapus)

## Teknologi yang Digunakan
- Laravel
- PHP
- MySQL

## Cara Menjalankan Aplikasi

1. Clone repository
2. Jalankan composer install
3. Jalankan php artisan migrate
4. Jalankan php artisan serve
5. Buka browser dan akses:
   http://127.0.0.1:8000
